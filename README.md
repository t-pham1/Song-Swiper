# Song-Swiper

## Prerequisites
- Python 3.13.3
- pip 25.1.1
- git 2.49.0

## Requirements
- Flask 3.0.2
- spotipy 2.23.0
- python-dotenv

## Setup
1. `git clone https://github.com/t-pham1/Song-Swiper.git`
2. `cd Song-Swiper`
3. `python3 -m venv venv`
4. `source venv/bin/activate`
5. `pip install -r requirements.txt`
6. Setup .env file (from Spotify dev dashboard)

## Usage
- `source venv/bin/activate`
- `ctrl + alt + n`
